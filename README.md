# 👁️ Computer Vision

A collection of hands-on Computer Vision assignments implemented in Python and Jupyter Notebook, covering core CV techniques from low-level image processing to deep learning-based vision models.

> 📚 Coursework — Master of Science in Computer Science, University of Maryland Baltimore County (UMBC)

## 🧪 Topics Covered

- Image processing & spatial filtering
- Edge detection (e.g. Sobel, Canny)
- Feature extraction & descriptor matching (e.g. SIFT, ORB)
- Object detection & recognition
- Deep learning-based vision (CNNs)
- Image segmentation

## 🚀 Getting Started

### Prerequisites

```bash
pip install numpy opencv-python matplotlib scikit-learn torch jupyter
```

### Run

```bash
git clone https://github.com/Gravity-2010/ComputerVision.git
cd ComputerVision
jupyter notebook
```

Then open any `hw*/` folder and run the notebooks in order.
